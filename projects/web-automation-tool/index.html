Web Automation with Notifications

Overview:

Python scripts to automate tasks on the browser. Useful for sites that do not have notifications or waiting lists. Goal is to avoid manual polling. Notifications on every platform thanks to an slack bot included.

Project State:

In development.

Used Technologies:

  • python for the scripts
  • selenium as the core library to control the browser
  • helium as an abstraction layer to interact with selenium
  • bolt for python as the framework for the slack bot usind websocket
  • slack API
